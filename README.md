# Сравнение сортировок на C

![C](https://img.shields.io/badge/Language-C-blue)

## Описание
Проект демонстрирует работу различных алгоритмов сортировки чисел с плавающей точкой (`float`) и позволяет сравнивать их по времени выполнения.  

### Реализованы алгоритмы:  
- **Selection Sort** — сортировка выбором  
- **Quick Sort (Хоара)** — быстрая сортировка  
- **Merge Sort** — сортировка слиянием  
- **Radix Sort для float** — поразрядная сортировка

### Программа позволяет:  
- Вводить собственный массив или генерировать случайный.  
- Выбирать сортировку для выполнения.  
- Отслеживать отсортированный массив и время работы алгоритма.  
- Проверять корректность всех алгоритмов, сравнивая результаты.

## Использование:
1. Запустите программу (на Windows или на Linux/Mac).  
2. Выберите режим работы:  
   - Введите `1`, чтобы ввести свой массив вручную.  
   - Введите `2`, чтобы сгенерировать случайный массив.  
3. Укажите количество элементов массива.  
4. Если выбрали ручной ввод, введите элементы через Enter.  
5. Выберите сортировку:  
   - Введите `1` для Selection Sort (сортировка выбором)  
   - Введите `2` для Quick Sort (быстрая сортировка Хоара)  
   - Введите `3` для Merge Sort (сортировка слиянием)  
   - Введите `4` для Radix Sort (поразрядная сортировка для float)  
6. Программа покажет:  
   - Отсортированный массив  
   - Время выполнения выбранной сортировки  
   - Проверку корректности: сравнение результатов всех алгоритмов
